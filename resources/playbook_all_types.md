# 骨架一：创始人故事深度

## 结构模板
```
【那个「一定要做这件事」的时刻】
用创始人人生轨迹中的一个关键决定时刻开场：
「[时间]，[创始人]做了一个外人看来有些奇怪的决定——[决定内容]。」

【他/她是谁：经历中的那些「预谋」】
不是简历，而是「哪段经历让他们适合做这件事」：
- 看起来不相关、实际至关重要的过去经历
- 他们做这件事时看到的，别人没看到的

【创业路上的「至暗时刻」】
（猎云网的差异化：愿意写失败）：
- 最接近放弃的那一刻发生了什么？
- 是什么让他们继续走下去？

【团队是怎么来的】
不只是「有N个联创」，而是：
- 他们为什么跟着创始人干？
- 团队里的代表性故事

【现在和接下来】
- 产品/公司当前状态
- 下一步最想做的事（直接问，让创始人说）
```

# 骨架二：早期项目发现

## 结构模板
```
【「为什么今天写这家公司」】
发现感先行：这家公司为什么值得在「早期」就被关注？

【一个非典型的用户故事】
找一个真实用户（不是创业者描述），讲他们怎么用这个产品

【产品本身用最简单的话说】
对外行人也能说清楚：这解决了什么问题，怎么解决的

【早期数据里的「真实信号」】
不要求大数字，要求「有增长趋势」的真实数据：
- 用户增长曲线（即使绝对值不大）
- 留存率/复购（比用户总量更重要）

【为什么这个时间节点是机会】
市场/技术/政策给的窗口在哪里？
```

# 骨架三：融资快讯报道

## 结构模板
```
【公司基本信息 + 融资信息（30字内）】
「[公司名]（[一句话描述]）近日完成[金额][轮次]融资，由[机构]领投。」

【公司做什么：用一个场景说明】
不是功能介绍，是「它帮[谁]解决了[什么具体问题]」

【投资人为什么看好】
尽力采访到投资方：他们的投资逻辑是什么（哪怕只有一两句话）

【创始人怎么说】
引用创始人对未来方向的表述（简短，2-3句话足够）

【简短的同赛道位置说明】
这家公司在赛道里是什么位置？有哪些主要竞争者？
```

# 骨架四：创业转型故事

## 结构模板
```
【转型前 vs. 转型后】
「[公司]在[时间]之前，是一家[描述]。现在它是[新描述]。这中间，发生了什么？」

【为什么要转】
原路走不通的真实困境（不要软化成「战略升级」）

【转型的代价】
转型不是免费的：放弃了什么、流失了什么、内部有什么阻力

【新方向的证明】
哪些数据或信号显示新方向是对的（哪怕还很早期）

【给其他创业者的启发】
一两句实话：他们从这次转型中学到的最重要的事
```

# 骨架五：创业生态报道

## 结构模板
```
【生态全景：今年这个生态发生了什么】
用5个关键事件/趋势构建今年的「生态图」

【从业者的真实心态】
采访多位创业者/投资人，呈现生态内部的真实情绪（乐观/焦虑/观望）

【新涌现的机会点】
哪些新机会/新需求在这个生态中正在形成

【系统性改善或恶化】
坦诚评估：这个生态今年变好了还是变难了，为什么

【值得关注的下一个周期】
基于现有信息，对未来12个月的谨慎展望
```
