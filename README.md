# 🚀 猎云网 Writer Skill

> 让 AI 以猎云网「创业者视角+时效快报」风格写科技创业报道的可复用 Skill。

## 简介

本 Skill 基于对猎云网数百篇科技创业报道的结构化解析，将猎云网「创业者第一视角+人物故事驱动+早期项目发现」的写作风格标准化为 AI 写作指令。

## 文件结构

```
lieyunwang-writer/
├── SKILL.md                          # 主技能文件
└── resources/
    ├── playbook_founder.md           # 骨架一：创始人故事深度
    ├── playbook_early_project.md     # 骨架二：早期项目发现
    ├── playbook_funding_news.md      # 骨架三：融资快讯报道
    ├── playbook_pivot.md             # 骨架四：创业转型故事
    └── playbook_ecosystem.md         # 骨架五：创业生态报道
```

## 猎云网核心写作风格

| 维度 | 特征 |
|------|------|
| 开场 | 创始人的「决定做这件事」的关键时刻 |
| 叙事 | 人物驱动，讲人而不只讲公司 |
| 时效 | 快速触达早期融资动态 |
| 语言 | 活泼有深度，不过于正式 |
| 禁忌 | 只写官方版本、忽视失败和挫折 |

## 核心写作哲学

- **创业者视角**：从「做这件事的人」出发，而非「公司」出发
- **发现早期**：关注还没被大家发现的早期项目，体现「发现感」
- **真实挫折**：愿意讲踩坑和失败，而不只写光鲜成绩
- **社区连接**：内容服务于创业社区，给创始人带去实际资源和连接

## 如何使用

```
使用 lieyunwang-writer，帮我用猎云网风格写一篇关于 [创业公司/创始人] 的报道
```
