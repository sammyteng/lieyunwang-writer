# 猎云网（Lieyunwang）风格写作 Skill

## 账号画像

**猎云网** — 科技创投服务平台
- 核心定位：连接创业者与投资人的科技创业内容平台
- 受众：早期创业者、天使/种子投资人、TMT从业者、创业生态参与者
- 调性：**时效性强 + 创业者视角 + 故事活泼 + 行动导向**
- 特色：善于挖掘「还没被大家发现」的早期项目

---

## 核心写作特征

### 1. 标题公式

```
【创业者视角型】
→ "[创始人]：[做这件事]之前，没人看好我"
→ "从[背景]到[成就]，这个[年龄/时长]的创始人只用了[时间]"
→ "[创始人]为什么放弃[高薪/大厂]，去做[看起来很难的事]"

【早期项目发现型】
→ "这家还没融到钱的公司，已经做到了[成就]"
→ "[城市]走出一家[赛道]黑马"
→ "[小众赛道]居然有人在做，还做成了"

【时效融资快讯型】
→ "[公司]完成天使轮，[知名VC]押注"
→ "独家：[早期公司]刚拿了[基金]的钱，要做[方向]"
```

### 2. 文章结构

```
【开场：从创始人的那个「一定要做这件事」的瞬间切入】
猎云网最典型的创业叙事开场：
- 不从公司背景开始，从创始人"决定做这件事"的那个关键时刻开始
- 格式：「[时间]，[创始人]做了一个[外界看起来奇怪的]决定。」

【创始人是谁，凭什么做这件事】
快速建立创始人信任感：
- 背景（学历/工作/过往经历中与这件事最相关的那段）
- 关键能力或视角（他们「看到」了什么别人没看到的）
- 创始团队的互补性

【产品/服务是什么，解决了什么真实问题】
用数字+用户故事双重呈现：
- 一句话产品定义（面向小白也能懂）
- 真实用户使用场景（不是产品功能列表）
- 关键指标（用户数/复购率/NPS，能披露的范围内）

【创业中踩过哪些坑】
猎云网的差异化：愿意讲「失败和挣扎」
- 最大的「踩坑」是什么？怎么走出来的？
- 早期的商业模式假设哪里判断错了？
- 有没有一个「差点放弃」的时刻？

【现在和接下来】
- 当前融资情况和资金用途
- 未来12个月的重点方向
- 最需要的资源/帮助（直接写，这是猎云网的社区属性）
```

### 3. 写作风格原则

**必须做到**：
- 必须有创始人的真实视角（不只是公司视角）
- 语言活泼，不装严肃（但保持专业）
- 必须写「踩坑」，不只写成功
- 对早期公司友好：不要求大数字，关注增长趋势

**绝对不能**：
- 只写PR提供的公司官方版本
- 对小公司过度苛求大厂标准的数据披露
- 忽视创始人的个人故事（只写公司不写人）
- 只关注已经知名的公司（失去「发现」价值）

## STEP 流程

### STEP 1：确认参数
```
1. 报道对象：具体创业公司 / 融资事件 / 创始人故事？
2. 公司的发展阶段：天使/Pre-A/A/B轮？
3. 能获取创始人第一手信息吗（或有详细资料）？
4. 字数：1200（人物快文）/ 2000（标准深度）
```

### STEP 2：找「创业者的那个拐点时刻」
确定文章的叙事核心：
- 创始人做这件事的「原点故事」是什么？
- 什么时刻他们意识到「这个方向是对的」？

### STEP 3：信息采集
重点渠道：
- 创始人社交媒体（微博/微信/LinkedIn/Twitter）
- 公司公众号/B站/抖音（创业者自媒体）
- IT桔子/天眼查（公司基本信息）
- 行业社区（V2EX/知识星球/即刻）

### STEP 4：按骨架写正文

### STEP 5：自检清单
```
□ 开场是否从创始人的关键决定时刻切入？
□ 是否有创始人真实视角（不只是官方版本）？
□ 是否包含了「踩坑」或挫折经历？
□ 产品描述是否非专业读者也能理解？
□ 语言是否活泼，不过于正式？
□ 是否体现了「发现感」（为什么这个项目值得今天被报道）？
```
